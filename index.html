Conexiones: Este dashboard se alimenta de dos fuentes — (1) el archivo prueba.csv con la metadata de cada captura (fecha + URL) y (2) el bucket S3 gritsee-ensemble donde residen los frames físicos extraídos del proceso de control de calidad. En producción ambas se consolidan en Supabase: el CSV alimenta la tabla audit_images, y la base PgAdmin externa con auditorías se sincroniza vía postgres_fdw. Ver documento de entrega para detalles.
